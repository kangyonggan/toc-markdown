# toc-markdown
根据markdown内容生成markdown格式的目录

## 效果
![效果](https://wx3.sinaimg.cn/large/0081fa71gy1glwwl3i6k2j30kb06ot8j.jpg)

## 安装
```
npm i toc-markdown
```

## 用法
```
import TocMarkdown from 'toc-markdown';

const toc = TocMarkdown.generateToc(content);
```
